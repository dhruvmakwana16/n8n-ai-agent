# 🤖 n8n AI Agent (Gemini)

This project is an AI Agent workflow built using **n8n** and **Google Gemini API**.

## 🚀 Features

* AI chatbot
* Conversation memory
* Tool integrations (Calculator / APIs)
* Automation ready workflow

## 🛠️ Tech Stack

* n8n
* Google Gemini API
* AI Agent node
* JSON workflow

## 📂 How to Use

1. Download the workflow JSON file
2. Import into n8n
3. Add your Gemini API key
4. Activate workflow

## 🎯 Use Cases

* Chatbot automation
* AI assistant
* Workflow automation
* Learning AI agents
